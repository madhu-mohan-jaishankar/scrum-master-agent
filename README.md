# scrum-master-agent

An AI-powered Scrum Master agent.


## Setup

Clone the repo and install the Bob skills in one command:

```bash
make install-skills
```

This copies the skills from [`.agents/skills/`](.agents/skills/) into [`.bob/skills/`](.bob/skills/), where Bob picks them up automatically.

## Available Skills

| Skill | Description |
|---|---|
| `spec-driven-development` | Creates a structured spec before any code is written. Use when starting a new project or feature, or when requirements are unclear. |
| `planning-and-task-breakdown` | Breaks a spec into ordered, verifiable tasks. Use when work feels too large to start or when implementation order isn't obvious. |

Bob will load these skills automatically once installed. You can verify they are active under **Bob Settings → Skills**.

## Project Structure

```
.agents/skills/          # Source of truth for custom skills (version-controlled)
.bob/skills/             # Runtime location Bob reads (generated by make install-skills)
Makefile                 # Install targets
```

## Re-installing After Pulling Changes

If a teammate updates a skill, re-run the install to pick up the changes:

```bash
make install-skills
```
